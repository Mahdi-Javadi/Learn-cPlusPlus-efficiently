Instruction

1- #include: for the input/output.
2- using std::cout: to prevent typing std:: each time we want to display something.
3- using std::cin : to prevent typing std:: each time we want to get anything from user.
4- using std::endl: to prevent typing std:: each time we want to add a new line in the output.
5- Each C++ program must have at least a main function with the format :

int main()
{
    
        //commands
    
        return 0;
}

6- Display the text in the output by using cout <<(extraction operator)
7- Declaration of the integer variable number.
8- Get an integer from the user and replace it by the value of the number by using cin and >> (insertion operator)
9- Display the number with a text and insert a new line by endl.
10- return 0, if the program works correctly.
